# Advanced Changelog

Figma + FigJam widget to track any changes in your stuff.

- 8 types of entries
- Manual sorting
- Editing protection
- Dark mode support

## What this widget is good for: 

- Design and components system updates
- Project and progress tracking
- Development changelogs
